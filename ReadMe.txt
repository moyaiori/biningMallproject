작성하기전에 한번씩 읽어봅시다.

1. index.jsp파일에 jstl, ajax등의 기본적인 태그나 기본폼은 적용해두었습니다.

2. 각자 작성시에 최상단에 최종 수정일, 작성자, 수정내용등을 작성합시다.
/**
	작성일 : 2015. 01. 01
	작성자 : 이광용
	내용 : (본 파일의 기능)
	수정내용 : 코맨트 수정
*/

3. 소스트리는 소스작성 -> 커밋 -> 풀 -> 푸쉬 순으로 합니다. 소스트리 애러방식 및 충돌시엔 저를 부릅니다.

4. 그외 소스는 자주 커밋 하고 푸쉬 하도록 합니다.
